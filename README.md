An Instagram scraper module for Silverstripe.

## Requirements

* Silverstripe Framework 4.x
* instagram-php-scraper 0.9
* phpfastcache 7.1

## Installation (with composer)

    $ composer require 3dgoo/silverstripe-instagram-scraper
